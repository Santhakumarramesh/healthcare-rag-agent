# Screenshots

Placeholder directory for UI screenshots.

To add screenshots:
1. Take screenshots of the live app at https://healthcare-rag-ui.onrender.com
2. Save as:
   - main-interface.png (chat with confidence scores)
   - medical-records.png (PDF upload tab)
   - architecture.png (architecture diagram screenshot)
3. Replace these placeholders

Alternatively, use the interactive HTML diagram at docs/architecture-diagram.html
