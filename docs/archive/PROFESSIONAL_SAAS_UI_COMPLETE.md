## Visual Refinements

### Cards
- Border radius: 12px (soft, rounded)
- Borders: 1px solid #D9E6F2 (subtle)
- Shadows: 0 2px 4px rgba(0,0,0,0.04) (very soft)
- Padding: 28px (generous spacing)
- Hover: Smooth transitions with accent color

### Status Cards
- Background: #F7FAFC (light gray)
- Indicator dots: 8px circles (green/amber/red)
- Uppercase labels with letter spacing
- Clean, minimal design

### Action Cards
- Gradient icon boxes (primary → accent)
- Top border animation on hover
- Descriptive text below title
- Professional, not playful

### Clinical Answer Cards
- Letter icons in rounded boxes (A, ✓, %)
- Section headers with icons
- Numbered bullet points for insights
- Color-coded confidence badges
- Expandable source citations
